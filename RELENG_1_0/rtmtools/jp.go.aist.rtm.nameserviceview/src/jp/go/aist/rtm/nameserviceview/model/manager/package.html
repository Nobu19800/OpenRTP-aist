RtcLink内部でのモデル(jp.go.aist.rtm.systemeditor.model)情報や設定情報にアクセスするためのマネージャが含まれる。 　　・NameServerManager 　　　　ネームサーバへのアクセス情報を管理しており、ネームサーバビューから使用される。 　　・PreferenceManager 　　　　RtcLinkの設定情報を管理する。 　　　　設定情報を取得するためには、このマネージャにアクセスすれば良い。 　　　　また、Eclipseの[Window]-[Preference]-[RtcLink]の画面では、このマネージャにアクセスして設定を編集している。 ダイアグラムのモデルオブジェクトを管理するマネージャは存在しない。 それは、jp.go.aist.rtm.systemeditor.ui.editor.SystemDiagramEditorに対してダイアグラムが、１対１に管理される。
